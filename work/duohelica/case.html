Duohelica

Duohelica was a site builder based off 1-click installs for the wysiwyg concrete5 CMS --before wysiwyg CMS's like squarespace took off. The project ended prematurely due to employment commitments at the time, but the branding work remains one of my favorite. The sales page and a github repo still remain.
